# Projetos da Faculdade

## RPG Call Of Cthulhu
* Descrição: Para a disciplina de Sistemas de Banco de Dados, eu e meu grupo criamos um RPG como trabalho da disciplina. Ele é baseado nos livros do escritor H.P. Lovecraft e utiliza um banco de dados em Postgres para organizar todas as mecânicas do jogo. Além disso, foi utilizado python para integrar os componentes do banco de dados com a interação do jogador no terminal do computador.
* Tecnologias utilizadas: PostgresSQL, Python.
* [Link do reposiório](https://github.com/luizfaria1989/2025.1-CallOfCthulhu)
